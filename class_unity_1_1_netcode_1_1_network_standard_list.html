Detailed Description

Represents a List<T> that is synchronized as a List<T>
This allows you to synchronize managed types objects.

Represents a List<TOriginal> that is synchronized as a List<TCast>
This allows you to:

Template Parameters
T
  • Synchronize managed types objects
  • Synchronize those objects with their <TCast> types (so it's serializer)
Template Parameters
TOriginalOriginal type of the objet (will be store with this one)
TCastCast type of the object (will be serialized with this one)
