Detailed Description

Represents a transition in a statemachine.

A transition can also be a self-transition that does not change state.

◆ Transition() [2/2]

CodeSmile.Statemachine.FSM.Transition.Transition ( String transitionName)

Creates a named transition that executes its actions if its conditions are satisfied.

If GotoState is non-null will also change state if conditions are satisfied.

The transition name should be used to annotate or summarize the purpose of the transition. The name is also instrumental for debugging and appears in diagrams.
