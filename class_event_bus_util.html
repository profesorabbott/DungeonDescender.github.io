static void EventBusUtil.Initialize ( )
static

Initializes the EventBusUtil class at runtime before the loading of any scene.

The [RuntimeInitializeOnLoadMethod] attribute instructs Unity to execute this method after the game has been loaded but before any scene has been loaded, in both Play Mode and after a Build is run. This guarantees that necessary initialization of bus-related types and events is done before any game objects, scripts or components have started.
